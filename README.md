🧠 「共生」外置大脑（Personal Knowledge Companion）· 使用说明书

共生是一个运行在本地的个人知识辅助工具。它不会帮你写任何文字，而是专注于帮你拆解零碎的灵感、整理跨领域素材，并在事后梳理知识。它更像是你的“外置大脑”，负责为你提供有结构的思考线索。

## 一、这是什么？
这个工具不会替你产出内容，而是捕捉你的闪光点、理顺跳跃的思维，把碎片打包成可用的素材。它和那些“先搭框架再填内容”的写作软件完全不同。你只要在里面随意倾倒破碎的随笔、视频创意，它会在后台帮你理清逻辑断层，然后按标签和时间轴，将这些碎片组织成有条理的记录。

## 二、谁适合用？
如果你属于下面这几类人，共生可能很适合你：

- **感知型创作者**：喜欢短篇、意识流或碎片化写作，思维活跃跳跃，厌烦动笔前先填一堆角色设定、大纲和事件表。
- **独立内容创作者 / 视频 UP 主**：想法多且碎，常同时推进小说、分镜、剪辑转场等跨界点子，想要一个无压力的灵感收集和分类箱。
- **深度控制型写作者**：觉得通用 AI 写不出你要的味道，坚决不用 AI 续写，但需要一位帮手来记住繁杂设定、理清前因后果。

## 三、有哪些功能？

### 1. 随笔主编辑器
在页面中央的编辑区，像用记事本一样写下你当下冒出的任何意识流片段、随笔。它提供纯粹的写作环境。停笔约 3 秒后，共生会自动扫描刚写下的文字，记住里面提及的人物言行或关键线索，并同步到右侧的感知日志。

### 2. 双轨灵感漫游（左侧面板）
当你有一个模糊的视频点子或小说冲突，却不知如何下手时，可以在左侧面板和共生对话，就像微信聊天一样。
- 如果你提到“视频、剪辑、转场、分镜”等词，它会自动切换为视听顾问，从画面意象和情绪曲线给你启发；如果是日常随笔，它就专注于帮你理清逻辑。
- 它只追问，不代笔——不会抛给你一堆废话正文，而是指出你跳跃思维中的因果断点，提出 2~3 个深度问题，促使你把思路走通。

### 3. 设定后置智能提取（右侧面板顶端）
写完一段意识流后，不必切换到单独的设定页面去填表。只需点击右侧的 “🧠 扫描随笔并提取设定”，共生就会通读你的随笔，把其中出现的新人物特质、特殊道具的物理规则、事件的前因后果抽取出来。你确认后，这些数据会自动追加到后台对应的设定库。设定会随着你的写作自然生长。

### 4. 资产打包一键投递（左右面板联动）
一段精彩的漫游对话结束后，或是写完一段满意的转场随笔，可以在下方的投递控制台选择目标容器（比如某篇文章、某个视频脚本，或纯灵感日志）。共生会自动提炼出一份结构化的精简摘要，把这段内容打包成一个素材单元，附上时间戳，追加到对应容器的末尾。

### 5. 大脑多维标签索引看板（右侧面板底端）
点击右侧底部的快捷标签（如 #视频转场、#人设演变），或直接输入关键词。系统会跨小说、视频等不同类别，把散落各处的历史随笔、对话摘要等，凡是带有相关标签的内容整合到一起，按时间轴呈现给你。

## 四、可以输出的结果是什么？
使用共生，你最终得到的不是 AI 批量生成的呆板正文，而是：

- **灵感摘要素材**：每一次零碎的火花都会被梳理成带有【核心意象】和【视听/叙事建议】的干净素材。
- **设定全景**：通过随笔后置提取，在后台悄然生长出来的人物性格演变线和世界观规则集。
- **思维笔记**：点击某个标签，得到的是一份跨时间、跨领域的完整思考记录。它把你过去所有破碎的构思汇总在一页，支持一键复制，随时查阅、投入实际制作。
